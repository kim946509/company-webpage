주요 핵심 System

우리엠텍이 보유한 핵심 시스템을 소개합니다

3D-CAD Automation System

UG-NX와 Library를 사용한 Full-3D 자동 설계 시스템으로, 효율적이고 정확한 설계를 가능하게 합니다.

  • Full-3D 자동 설계 구현
  • UG-NX 기반 설계 자동화
  • Library 시스템 활용
  • 설계 시간 단축 및 정확도 향상

CAE Injection Molding Analysis System

설계 전 CAE를 통한 사출성형 해석 시스템으로, 제품의 품질과 생산성을 향상시킵니다.

  • 사출성형 해석
  • 유동 해석
  • 냉각 해석
  • 변형 예측 분석

CAM Automation System

MS & NX CAM을 활용한 One-Stop 자동화 시스템으로, 효율적인 가공 공정을 실현합니다.

  • One-Stop 가공 프로세스
  • 밀링 레퍼런스 DB 활용
  • 자동 전산화 시스템
  • 가공 시간 최적화

CMM & QMMS System

3차원 정밀측정기를 활용한 품질관리 시스템으로, 높은 정확도의 측정과 품질 보증을 제공합니다.

  • 3차원 정밀측정
  • 입고품 정밀 검사
  • 가공 완료품 측정
  • 품질 데이터 관리

Steam-Mold System

고광택 외관 제품을 위한 특수 금형 기술로, 뛰어난 표면 품질을 실현합니다.

  • 고광택 표면 처리
  • 외관 품질 향상
  • 안정적인 생산성
  • 에너지 효율 최적화

  • 친환경 생산 기술 연구

사출 압축 기술

(특허 제10-1352434호)
(특허 제10-1371351호)

Glass(유리)로 되어 있는 자동차의 창문을 플라스틱 소재(PC)로 대체할 필요성이 요구되고 있다. 그 이유는 무거운 유리에서 가벼운 플라스틱으로 대체하면 중량감소(50%)에 의해 자동차에서 발생하는 공해물질 및 탄소배출의 저감을 실현할 수 있다. ( ⁕비중 : 유리 -2.4, 플라스틱 -1.2 )
